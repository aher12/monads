# ЛР1: Монады — Вариант 17. Турнирная таблица

## Меню
```
Меню построено по древовидной структуре (MenuTreeNode + MenuLeaf).
Пункты нумеруются автоматически через zipWithIndex. Добавление нового пункта сводится к добавлению MenuLeaf в Seq.
```
## Пункты меню
```
Add team — добавить команду в турнир

Record match result — записать результат матча (две команды, счёт)

Show table — вывести турнирную таблицу с сортировкой по тай-брейку

Show matches — показать все сыгранные матчи

Next round — перейти к следующему туру

Reset tournament — сбросить турнир до начального состояния

Exit — выход с итоговой статистикой
```
## Конфигурация (Reader)
```
Из TournamentConfig читаются:

pointsForWin / pointsForDraw / pointsForLoss — очки за исходы

tieBreak — правило тай-брейка (GoalDifference, GoalsScored, HeadToHead, GoalDiffThenScored)

maxRounds — максимальное число туров

forbidRematch — запрет повторной игры команд в одном туре
```
## Логирование (Writer)
```
Лог пишется через Writer[Vector[String], A]:

Запись результата матча

Начисление очков команде

Изменение позиции в таблице
```
## Состояние (State)
```
TournamentState содержит:

teams: List[String] — список команд

matches: List[MatchResult] — все сыгранные матчи

currentRound: Int — текущий тур
```
## IO

```
Консольное взаимодействие с пользователем 
- добавить результат матча
- вывести таблицу
- добавить команду
- показать все матчи
- следующий тур
- сброс турнира
```
